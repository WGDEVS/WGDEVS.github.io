Swippy Bird

A game where you play as a bird and avoid geometric shapes by swiping the screen. How far can the bird make it?

Features:

  • Movement of the bird with touch screen input.

  • Beautiful HUD drawn using android canvas.

  • Sharing the images of completed games with android's built in share event.

  • Comprehensive leaderboard and achievement system with google play integration.

  • Lively sound effects using android's built in soundpool class.

  • Interstitial ads with leadbolt integration.

  • Ability to resume paused games with careful management of the andoid lifecycle.

  • Procedural generation of the geometric shapes with recursion.

Integer Sorter

A program that contains distinct functions for sorting a list of integers. The program recieves a list of integers, sorts the list using a method specified by the user and prints the sorted list to the screen.

Features an implementation of:

  • Mergesort

  • Insertion Sort

  • Quicksort

  • Bubblesort

  • Binary Search

Router Simulator

This program simulates a network consisting of routers. It features a command line interface to build and maintain the network and to display information about the network.

Features:

  • Finds the best route between two routers using dijkstra's algorithm

  • Quickly looks up important data about the network using binary search

  • Saves and loads the network to an external file using built-in python functions

  • A comprehensive list structure that quickly and efficiently stores information about the network.

  • Packet tracer inspired names for the commands.

Sea Battle: Custom Mission

Classic game of sea battle, with options to customize gameplay.

Features:

  • Very modular game items (ships/planes/missiles) made possible with object oriented programing

  • Efficient initialization of games and handling of turns using polymorphism

  • Item classes made simpler with inheritance

  • Gameplay against other players on the same LAN using Microsoft's system.net namespace

  • A very clever AI to play against that quickly places ships in an unpredictable pattern and fires in a way that will never hit the same location twice
